# LivrosFavoritos
Desenvolvido por: Arthur Lima, Felipe Lanza, Lucas Gouveia, Murilo Almeida e Won Woo Chung

O aplicativo Lista de Livros foi desenvolvido com o objetivo de apresentar ao usuário uma coleção de obras literárias de forma simples, intuitiva e visualmente agradável. A interface apresenta uma lista organizada contendo capas dos livros, títulos e nomes dos autores, tornando a navegação prática e agradável.



# Funcionalidades Principais

Exibição de livros com imagens: Cada item da lista exibe a capa original do livro para facilitar reconhecimento visual.

Informações essenciais: O usuário visualiza o título e o autor de cada obra rapidamente.

Interface minimalista e responsiva: A tela se adapta a diferentes dispositivos Android, mantendo boa legibilidade e design limpo.

Scroll contínuo: Toda a lista pode ser percorrida de forma suave através de uma ScrollView.



# Tecnologias Utilizadas

Kotlin — Linguagem principal do desenvolvimento Android moderno.

Android XML Layout — Responsável pela estrutura visual e disposição dos componentes.

Android Studio — Ambiente de desenvolvimento utilizado para criação, testes e ajustes.

Arquitetura da Tela

A tela principal (activity_main.xml) contém:

Um título (“Lista de Livros Books”)

Uma lista vertical composta por itens de livro, cada um contendo:

ImageView para a capa

TextView para o título

TextView para o autor

Estrutura organizada em LinearLayouts com orientação vertical e horizontal.



# Objetivo do App

O app tem como objetivo servir como um exemplo funcional de interface Android para iniciantes ou estudantes, facilitando o aprendizado sobre:

Estruturação de layouts,

Uso de imagens no Android,

Exibição de listas estáticas,

Estilização simples e clara.

Possíveis Expansões Futuras

Implementação de RecyclerView para listas dinâmicas.

Tela de detalhes do livro.

Função de favoritos.

Busca por título ou autor.

Integração com API externa de livros.
